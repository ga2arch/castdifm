Castfdifm
=============
### Cast di.fm station to the chromecast

Installation
------------

``` bash
$ git clone https://github.com/ga2arch/castdifm
$ cd castdifm
$ npm install
```

Usage
-----
```bash
$ node castdifm.js deephouse [--key=<listen_key>]
connected
$ node castdifm.js --stop
connected
```
